## Trabajo Práctico 0 

Este trabajo tiene como objetivo familiarizarse con el entorno de desarrollo en lenguaje C, incluyendo la instalación de herramientas necesarias como el compilador GCC, Make, Flex y Bison.
Para ello se desarrolla un programa simple en lenguaje C que imprime un mensaje por pantalla ("Hola mundo"), con el fin de verificar el correcto funcionamiento del entorno.
